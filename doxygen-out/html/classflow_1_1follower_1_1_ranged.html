Detailed Description

template<typename DispatchT, typename LockPolicyT = NoLock, typename ContainerT = DefaultContainer<DispatchT>, typename QueueMonitorT = DefaultDispatchQueueMonitor>
class flow::follower::Ranged< DispatchT, LockPolicyT, ContainerT, QueueMonitorT >

Captures one one element before the capture range lower bound; one element after the capture range upper bound. 

   All elements in between are also captured. All older elements are removed.
