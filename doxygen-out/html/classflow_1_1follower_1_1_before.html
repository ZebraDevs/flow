Detailed Description

template<typename DispatchT, typename LockPolicyT = NoLock, typename ContainerT = DefaultContainer<DispatchT>, typename QueueMonitorT = DefaultDispatchQueueMonitor>
class flow::follower::Before< DispatchT, LockPolicyT, ContainerT, QueueMonitorT >

Captures all elements before the capture range lower bound, minus a delay period. 

   Once at least a single element is available after said sequencing boundary.
   All of the captured elements are removed.
