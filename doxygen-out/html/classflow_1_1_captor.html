Detailed Description

template<typename CaptorT, typename LockableT, typename QueueMonitorT>
class flow::Captor< CaptorT, LockableT, QueueMonitorT >

Default constructor.

CRTP-base for input capture buffers.

Note
Captor implementation which uses data waits/locks on Captor::capture for use in asynchronous, multi-threaded code.

User may include any of the following for various capture implementations

  • #include <flow/captor/lockable.hpp>
  • #include <flow/captor/nolock.hpp>
  • #include <flow/captor/polling.hpp>
Template Parameters
CaptorTCRTP-derived Captor type
LockableTa TimedLockable (https://en.cppreference.com/w/cpp/named_req/TimedLockable) object; specializations are available which replace LockableT with NoLock or PollingLock
QueueMonitorTobject used to monitor queue state on each insertion; used to precondition capture
