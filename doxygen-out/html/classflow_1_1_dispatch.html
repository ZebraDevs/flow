Detailed Description

template<typename StampT, typename ValueT>
class flow::Dispatch< StampT, ValueT >

Dispatch data wrapper.

Custom "dispatching" objects may be used with this library in lieu of the standard Dispatch template provided in this file
Custom dispatch types must also specialize the DispatchTraits helper struct, which is used to provide core stamp_type and value_type to other dependent objects throughout the library.

Template Parameters
StampTsequencing stamp type; used for data ordering (e.g. time, sequence number, etc.)
ValueTdata value type; must be copyable
