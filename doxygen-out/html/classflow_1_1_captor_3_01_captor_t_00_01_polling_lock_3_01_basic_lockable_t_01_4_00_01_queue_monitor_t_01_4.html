Detailed Description

template<typename CaptorT, typename BasicLockableT, typename QueueMonitorT>
class flow::Captor< CaptorT, PollingLock< BasicLockableT >, QueueMonitorT >

Dispatch container constructor.

Parameters
containercontainer object with some initial state
queue_monitorcustom implementation for checking the state of the queue and preconditioning capture
Note
Initializes data capacity with NO LIMITS on buffer size
Implements a simple locking policy using a BasicLockableT which protects data input/output, meant for polling with Captor::capture.
